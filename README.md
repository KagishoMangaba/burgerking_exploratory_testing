🍔 Burger King Mobile App – Exploratory Testing Report

## 📌 Overview

- This repository documents exploratory testing performed on the Burger King mobile application.
- The purpose of this exercise was to:
- Identify usability issues
- Evaluate input validation behavio
- Assess data integrity risks
- Practice structured defect documentation
- Testing was conducted from a user perspective without access to internal requirements or source code.

## 🧪 Scope of Testing

- Profile Section
- Input Validation (Name & Phone Number Fields)
- Navigation / UI Clarity

Testing Type:

- Exploratory Testing
- Functional Testing
- Usability Testing
