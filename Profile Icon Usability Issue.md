- Application: Burger King Mobile App
- Module: Navigation / UI
- Severity: Medium
- Type: Usability Defect

🐞 Defect Title

- Profile icon lacks clear identification and causes user confusion.

📌 Description

- The profile button/icon does not clearly indicate that it redirects to the Profile section.
- There is:
- No clear label.

- No visual cue suggesting user account/profile.
- No hover text or tooltip.
- Users must click it to discover its purpose.

🔁 Steps to Reproduce

- Open app homepage.
- Observe top navigation area.
- Attempt to identify profile access point without clicking.

❌ Actual Result

The profile icon appears as a generic or unclear button.
Users cannot confidently predict its destination.

✅ Expected Result

- Profile icon should:
- Use a universally recognized profile symbol (e.g., user avatar).
- Include label “Profile” or similar.
- Provide intuitive UX guidance.

🎯 Impact

- Poor user experience.
- Increased cognitive load.
- Reduced accessibility and usability clarity.
